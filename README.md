# prefix-notation-calculator

Calculates prefix notation arithmetic expressions with an intuitive recursive solution that matches the way humans reason about the expression.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.10. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
